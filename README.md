# ApexPlanet Data Analytics Internship – Final Capstone

## Project Overview
This project analyses a 2025 sales transactions dataset containing 1,000 orders and 12 fields. The objective was to clean the data, identify sales patterns, build an interactive Power BI dashboard, and validate a business question using statistical testing.

## Tools Used
- Python
- Pandas
- Power BI
- SciPy
- GitHub

## Key Tasks Completed
1. Data Cleaning
   - Filled 20 missing Age values using the mean age.
   - Replaced 13 missing City values with "Unknown".
   - Checked for duplicate records.
   - Created a Discount field for analysis.

2. Exploratory Data Analysis
   - Analysed orders by category, city, product, and gender.
   - Electronics was the highest-order category.
   - Mobile was the best-selling product.
   - Patna had the highest order count among the top five cities.

3. Interactive Dashboard
   - Created Power BI visuals for total sales, total orders, category sales, city sales, gender distribution, and monthly sales.
   - Added slicers for City, Category, and Gender.

4. Statistical Validation
   - Performed One-Way ANOVA to test whether age group affects total sales.
   - Result: p-value = 0.989.
   - Conclusion: Customer age group did not have a statistically significant effect on total sales.

## Key Recommendation
The business should focus on category-level and city-level performance rather than age-based targeting.
